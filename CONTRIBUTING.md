# Contributing

Thank you for helping expand this list of missing O'Reilly-style books!

## How to add a book

1. Create a new Markdown file in the `books/` folder.
2. Name it using lowercase letters and dashes, e.g., `domain-driven-infrastructure.md`.
3. Use the following template inside the file:

## Book Title

**Proposed animal:**  
**Audience:**  
**Thesis:**  
**Why missing:**  
**Sample back cover text:**  
**Links / references:**
